# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `sandbar-cli` or any other Sandbar
component, please report it to **security@sandbar.cloud**. Do not open a public
GitHub issue.

When reporting, please:

- Provide sufficient detail for us to reproduce the issue.
- Allow reasonable time for us to address the vulnerability before public
  disclosure.
- Do not access or modify data belonging to other customers.

We aim to acknowledge reports within two business days and to keep you
informed as we investigate.

## Supported Versions

Security fixes are applied to the latest released minor version of
`sandbar-cli`. Older versions are not patched — please update to the latest
release before reporting.
